# Lab19 - Gestion de Notes

Application Android simple pour la gestion de notes.

## Auteur
CHARRAJ Mouad (ZERO-XR7)

## Technologies
- Java
- Room Database
- ViewModel & LiveData
- ConstraintLayout & CardView

## Fonctionnalités
- Ajout de notes avec titre et description.
- Affichage de la liste des notes.
- Suppression d'une note via un appui long.
- Suppression de toutes les notes.
